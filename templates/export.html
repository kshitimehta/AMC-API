Itineraries Table gives information for all the reservations in the system along with a few details. The details include the itinerary id, reservation number, guest id associated with each reservation, group sizes for each visit, the travel dates, buildings visited by each trip, distances travlled and the group type code.


The Emissions Table gives information about the carbonfootprint by guest travel for all the itineraries over the years. It contains details about the travel dates, day of the week, zipcodes travelled from, buildings visited at AMC, group sizes and the 4 Green House Gas Emissions models.


The AMC Buildings Table gives as output all unique facilities at AMC. It includes details about the airports nearest to that facility and the distances to those airports.


The Distance Lookup Table would output all unique distances between origin location of guests and destination location at AMC. It includes details about the building codes, city, state, country and zipcode.


The Processing Error Table would output all errors related to missing values and the invalid values in the raw data by year. This will allow AMC to be able to manipulate the emissions values in a distributed way to account for the errors.
